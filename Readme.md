# Home Service App

Aplikasi mobile berbasis Android untuk pemesanan jasa servis rumah tangga seperti AC, kulkas, kipas angin, mesin cuci, dan perangkat elektronik lainnya.

---

## 📌 Deskripsi Proyek

Home Service App dibuat untuk memudahkan pengguna dalam mencari teknisi servis terpercaya secara cepat dan praktis melalui smartphone.

Pengguna dapat melakukan pemesanan jasa, memilih jadwal servis, melihat status pesanan, serta berkomunikasi dengan teknisi melalui fitur chat.

---

## 👥 Target Pengguna

- Masyarakat umum / pemilik rumah
- Mahasiswa / anak kos
- Teknisi servis elektronik
- Admin pengelola sistem

## ⚙️ Cara Menjalankan Project

### Android App

1. Buka folder `mobile-app` di Android Studio
2. Sync Gradle
3. Run Emulator / Android Device

### Backend API

1. Masuk folder `backend-api`
2. Configure PostgreSQL
3. Jalankan:

```bash
mvn spring-boot:run
```
